# CNN 2026

Site événementiel premium développé avec Next.js, TypeScript et Tailwind CSS.

## Lancer le projet

```bash
npm install
npm run dev
```

Puis ouvrir `http://localhost:3000`.

## Pages principales

- `/`
- `/the-concept`
- `/the-draft`
- `/mentors-program`
- `/white-paper`
- `/press-partners`
- `/apply`
- `/cohort-2026`
- `/squads`
- `/faq`
- `/contact`

## Configuration éditoriale

La phase du projet se règle dans `src/data/site.ts` avec `isAfterEvent`.
